# 🎹 DRGR SYNTH STUDIO | CLSK Miner

## 🚀 Играй, создавай музыку и зарабатывай токены CLSK!

**DRGR Synth Studio** — это революционное Telegram Mini App, которое объединяет музыкальное творчество и криптовалюту. Создавай мелодии на профессиональном синтезаторе и зарабатывай токены CLSK за каждую сыгранную ноту!

### ✨ Основные функции:

🎵 **Профессиональный синтезатор**
- 4 типа волн осциллятора (Sawtooth, Square, Sine, Triangle)
- ADSR огибающая для точного контроля звука
- 3 типа фильтров (Lowpass, Highpass, Bandpass)
- Delay эффекты с настройкой времени и обратной связи
- Полноценная клавиатура пианино (2.5 октавы)
- Спектральная визуализация в реальном времени

💎 **CLSK Token Mining**
- Зарабатывай 5 CLSK за каждую сыгранную ноту
- Токены начисляются мгновенно
- Интеграция с TON blockchain
- Связь с токеном Classik (@classikdeth)

🎨 **Современный интерфейс**
- Киберпанк дизайн с неоновыми эффектами
- Адаптивная визуализация звука
- Интуитивное управление
- Полная поддержка мобильных устройств
